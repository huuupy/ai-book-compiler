# 智能文章采集与PDF生成系统

## 项目概述

一个现代化的Web应用，实现"自动检索、整理、排版、输出PDF"的全流程自动化。用户可以通过可视化界面采集网页内容、AI智能清洗整理、自定义排版风格、一键导出精美PDF。

## 技术栈

| 模块 | 技术选型 | 说明 |
|------|---------|------|
| 前端框架 | Next.js 14 + React 18 | App Router架构 |
| 样式方案 | Tailwind CSS 3.4 | 原子化CSS |
| 爬虫引擎 | Crawlee (Node.js) | 强大的网页采集能力 |
| AI处理 | LangChain + DeepSeek API | 内容清洗与结构化 |
| PDF生成 | Playwright | 无头浏览器导出PDF |
| 状态管理 | Zustand | 轻量级状态管理 |
| 数据库 | SQLite (本地) | 文章数据存储 |

## 功能模块

### 1. 文章采集模块
- **URL输入**: 支持单个或批量URL输入
- **网站选择**: 预设支持的网站模板（知乎、Medium、微信公众号等）
- **采集配置**:
  - 代理设置
  - 请求间隔
  - 超时时间
  - 最大并发数
- **进度展示**: 实时显示采集进度和状态

### 2. AI内容处理模块
- **内容清洗**: 去除广告、导航栏、页脚等干扰内容
- **格式转换**: 将HTML内容转换为标准Markdown
- **摘要生成**: 自动生成文章摘要
- **标签提取**: 智能提取文章关键词/标签
- **多语言支持**: 支持中英文内容处理

### 3. 文章管理模块
- **文章列表**: 卡片式展示所有已采集文章
- **搜索筛选**: 支持标题、标签、日期搜索
- **批量操作**: 勾选多篇文章进行批量操作
- **文章预览**: 点击查看文章详细内容
- **分类管理**: 创建文件夹/分类管理文章

### 4. 排版引擎模块
- **模板选择**: 预设多种排版模板
  - 简约风格
  - 杂志风格
  - 学术风格
  - 自定义
- **样式配置**:
  - 字体选择（中文/英文）
  - 字号大小
  - 行高设置
  - 页边距
  - 主题色选择
- **布局选项**:
  - 单栏/双栏
  - 是否显示目录
  - 页眉页脚设置
  - 页码位置
- **实时预览**: 所见即所得的预览效果

### 5. PDF导出模块
- **批量导出**: 将多篇文章合并为一个PDF
- **导出选项**:
  - 纸张大小 (A4/Letter)
  - 导出质量 (高/中/低)
  - 是否包含封面
  - 是否包含目录
- **下载管理**: 显示导出进度，支持中断

## 数据模型

### Article (文章)
```typescript
interface Article {
  id: string;
  url: string;
  title: string;
  author?: string;
  publishDate?: string;
  summary: string;
  content: string; // Markdown格式
  tags: string[];
  coverImage?: string;
  source: string;
  status: 'pending' | 'processing' | 'completed' | 'failed';
  createdAt: string;
  updatedAt: string;
}
```

### Collection (合集)
```typescript
interface Collection {
  id: string;
  name: string;
  description?: string;
  articleIds: string[];
  template: 'simple' | 'magazine' | 'academic' | 'custom';
  settings: PrintSettings;
  createdAt: string;
  updatedAt: string;
}
```

### PrintSettings (打印设置)
```typescript
interface PrintSettings {
  fontFamily: string;
  fontSize: number;
  lineHeight: number;
  margin: number;
  layout: 'single' | 'double';
  showToc: boolean;
  showCover: boolean;
  showPageNumber: boolean;
  pageSize: 'A4' | 'Letter';
  primaryColor: string;
}
```

## API接口设计

### 文章管理
- `GET /api/articles` - 获取文章列表
- `GET /api/articles/:id` - 获取单篇文章详情
- `POST /api/articles` - 创建文章
- `PUT /api/articles/:id` - 更新文章
- `DELETE /api/articles/:id` - 删除文章

### 采集任务
- `POST /api/crawl` - 开始采集任务
- `GET /api/crawl/status/:jobId` - 获取采集状态
- `POST /api/crawl/batch` - 批量采集

### AI处理
- `POST /api/ai/process` - 处理单篇文章
- `POST /api/ai/batch` - 批量处理

### 导出
- `POST /api/export/preview` - 生成预览
- `POST /api/export/pdf` - 导出PDF
- `GET /api/export/status/:jobId` - 获取导出状态
